# G-Débat

Le nom de notre projet est V/S. Il représente les différentes opinions dans un débat:
    V pour les personnes contre une idée
    S pour celles qui sont pour
    / pour celles qui sont neutres

Ce nom reflète parfaitement l’objectif de notre projet qui est créer une plateforme de débat en ligne où chaque utilisateur peut donner son avis sur un sujet précis, voter pour, contre ou neutre, et même créer ses propres sujets de débat.

# Comment le site fonctionne
### Création de comptes: 
Les utilisateurs peuvent créer un compte pour participer aux débats ou en créer de nouveaux. Lors de l'inscription ils doivent fournir:
  -Leur adresse e-mail, numéro de téléphone, date de naissance, nom, et prénom.
  -Une sélection des catégories de débats préférées (politique, science, culture, etc.).
### Participation aux débats: 
Chaque sujet de débat propose trois choix de vote : pour, contre, ou neutre.
Les débats sont classés par catégories, ce qui permet aux utilisateurs de s’exprimer sur des thèmes qui les intéressent. Il y a une durée limité pour chaque débat.
### Création de sujets de débat: 
Les utilisateurs peuvent proposer des sujets en précisant la catégorie (politique, scientifique, etc.). Ils doivent également définir une durée limite pour le débat.
### Résultats des débats: 
Une fois le débat terminé, le site calcule automatiquement le nombre des personnes qui ont changé leur choix. Le côté qui a reçu le plus de personnes est affiché comme le gagnant du débat.
### Explications pour les nouveaux visiteurs: 
À l’accueil, le site affiche un résumé de son fonctionnement et de son objectif pour guider les nouveaux utilisateurs.


## Utiles
Pour réaliser ce projet, nous allons utiliser : Python, Java, HTML/CSS. 

## Objective
L’objectif de cette plateforme est d’offrir un espace accessible et structuré pour débattre de manière constructive, tout en respectant les règles de tolérance. Nous espérons encourager la discussion et permettre aux utilisateurs de mieux comprendre différents points de vue.
